tempdir()

tempdir(string|null  $dir = null, string  $prefix = 'tmp_', integer  $mode = 448, integer  $maxAttempts = 1000) : string|boolean

Creates a random unique temporary directory, with specified parameters, that does not already exist (like tempnam(), but for dirs).

Created dir will begin with the specified prefix, followed by random numbers.

Parameters

string|null $dir

directory under which to create temp dir. If null, default system temp dir will be used
string $prefix

String with which to prefix created dirs.
integer $mode

Octal file permission mask for the newly-created dir. Should begin with a 0.
integer $maxAttempts

Maximum attempts before giving up (to prevent endless loops).

Returns

string|boolean —

Full path to newly-created dir, or false on failure.

See also
https://php.net/manual/en/function.tempnam.php
http://stackoverflow.com/questions/1707801/making-a-temporary-dir-for-unpacking-a-zipfile-into
